 Research @ DSL

If we knew what we were doing, it wouldn't be called research. Would it? - Albert Einstein.

Ongoing Projects

  • Development of laser based 3D ranging device (LiDAR) for high resolution facial depth data acquisition and recognition.
    [Sponsored by SERB, DST as a part of the IMPRINT-II program]
    Following are some of the works that are being carried out as a part of this project:
    • CMOS IC design for laser and MEMS control.
    • MEMS micro-mirror for laser beam steering.
    • Machine learning algorithms for object detection with facial-depth data.
  • Assessment of stress in bank employees using Machine Learning and Virtual Reality:
    [Sponsored by Bank of Baroda]
    • Funded by Bank of Baroda, this project aims at studying the stress responses of bank employees. The stress responses are generated by subjecting the employees to a stressful environment using Virtual Reality. Machine Learning algorithms are employed on the generated responses to identify the factors that change when stress levels are increased or decreased.
  • Acquisition and Processing of EEG Signals:
    • This project is aimed at study of different techniques for classification of Electroencephalograph (EEG) signals. The project also explores various pre-preocessing techniques like artifact removal and feature extraction of EEG signals.
  • Low Power Analog Circuits for ML applications:
    • This work mainly focusses on building low power analog circuits for image and speech recognition applications. Mulitple machine/deep learning algorithms, some of them novel, have been employed in this work so as to achieve a classification accuracy at par with the existing models. The hardware design of this project is being done on the Cadence Virtuoso tool using UMC 130nm technology.
